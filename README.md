O objetivo é praticar lógica, sintaxe da linguagem e resolução de problemas através de exercícios variados.

- Lógica de programação
- Estruturas condicionais
- Laços de repetição
- Funções
- Listas e dicionários
- Exercícios matemáticos
- Tratamento de erros

## 📌 Objetivo

Praticar Python e desenvolver lógica de programação através de exercícios e desafios simples.
